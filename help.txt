Start Both Services
You have two options:
Option A - Using Concurrent Start (Recommended)

    npm run dev

This will start both the bot and dashboard simultaneously.
Option B - Start Separately (If Option A doesn't work)

Terminal 1:
 node index.js

Terminal 2:
cd v0.0.1-dashboard
   npm run dev
